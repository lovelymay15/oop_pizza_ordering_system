# Pizza Ordering System

A Java desktop application developed to demonstrate Object-Oriented Programming (OOP) principles. This system provides a digital interface for customizing pizza orders and calculating totals in real-time.

## Project Overview
The Pizza Ordering System streamlines the order process by managing pricing logic and customization options within a cohesive workflow. It was designed with a focus on modular code and user-friendly interaction.

## Key Features
* Customization: Options for various pizza sizes, crust styles, and toppings.
* Automated Pricing: Dynamic cost calculation based on user selections.
* User Interface: A structured GUI built for a seamless ordering experience.
* OOP Standards: Implementation of classes and objects to manage system logic.

## Technical Stack
* Language: Java
* Interface: Java Swing
* Environment: Apache NetBeans
